Django est un framework web Python de haut niveau qui encourage un développement rapide et propre avec un design pragmatique. L'ORM Django permet d'interagir avec la base de données en utilisant du code Python au lieu d'écrire du SQL. ASGI est l'interface de serveur asynchrone de Django qui permet de gérer des connexions WebSocket. On peut sécuriser une API Django REST avec JWT, les permissions RBAC, le throttling et HTTPS. Le middleware Django est un système de hooks qui traite les requêtes et réponses globalement.
